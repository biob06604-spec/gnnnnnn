# [Maize-GNN]

##  项目简介
Maize-GNN是一个深度学习项目，旨在利用 图神经网络 (Graph Neural Networks, GNN) 挖掘玉米中的潜在性状与基因的关联。
本项目构建了一个包含Gene和GO Term的异构图，整合了PPI和 GO 功能注释信息。通过链路预测 (Link Prediction) 任务和鲁棒的训练策略，模型能够学习基因的高维向量表示，并基于向量相似度推荐潜在的候选基因。
